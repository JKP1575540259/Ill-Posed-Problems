The folder contains 5 function files. 
The "AdapIni.m" is for initial adaptive regularization without iteration; 
The "AdapIter.m" is for iterative adaptive regularization; 
The "DeterAlpha.m" is for determining the regularization parameter used in "AdapIni.m" ; 
The "DeterAlphaIter.m" is for determining the regularization parameter used in "AdapIter.m" ; 
The "RegParm.m" is used in  "DeterAlpha.m" and "DeterAlphaIter.m";
